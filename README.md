# Laravel + Inertia + React 学習プロジェクト

このリポジトリは、Laravel 11、Inertia.js、Reactを使用したモダンなWebアプリケーション開発の学習用プロジェクトです。

## 概要

- Laravel 11
- Inertia.js
- React
- SQLite

## 環境構築手順

```bash
# プロジェクトのインストール
composer install
npm install

# データベースの作成
php artisan migrate

# 開発サーバーの起動
php artisan serve
npm run dev

# ブラウザでアクセス
http://localhost:8000
```
